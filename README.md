# jogodetetive
O jogo usa um array de objetos com perguntas, cada uma contendo título, texto, resposta correta (isFake) e explicação. O controlador usa estados para alternar telas e variáveis para índice atual e pontuação. Funções iniciam o jogo, processam respostas comparando valores booleanos e avançam perguntas. Renderiza componentes conforme o estado.
